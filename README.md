#  Musicana - A place for your favorite music

A simple music streaming web application where users can play songs, switch between tracks, and control the music with an easy-to-use interface.

## Features

*  Play and pause music
*  Next and previous song
*  Volume control
*  Multiple playlists
*  Responsive design

## Built With

* HTML
* CSS
* JavaScript
* Node.js
* Express.js

## Installation

```bash
git clone https://github.com/dikshaa-20/Musicana.git
cd Musicana
npm install
node server.js
```

Open your browser and go to:

```text
http://localhost:3000
```
